# Eldoria Farm Master Palette v1

Source contract: [`docs/VISUAL_ASSET_CONTRACT.md`](../VISUAL_ASSET_CONTRACT.md), Section 4 (Palette And Lighting) and the [Stardew-Caliber Visual Research](../research/visual-design/STARDEW_CALIBER_VISUAL_RESEARCH_2026-07.md) Finding 3.

## Purpose

Section 4 of the contract requires that a sub-palette's **actual hex values** be locked "in a small swatch reference alongside that asset's PR so later assets in the same family cannot silently drift." The other Phase 2 target files name palette families (`forest`, `wood_leather`, `metal`, `arcane`) but do not fix their colors. This file is that lock: an authoritative, versioned, machine-checked set of hex values that any farm-environment source art — including art generated by an external tool — must draw from.

## Scope

The farm environment kit only. Character sub-palettes (`skin_hair`, `ui_neutral`, etc.) are locked separately when their own production art lands and are deliberately not defined here. Specification only — no art, no runtime behavior.

## Families

Ramps are ordered dark → light. The artist/generator applies the contract's consistent upper-left key light (highlights upper-left, shadows down-right); do not bake a scene-level ambient tint into source art (Section 14 applies that once, in code).

## Pinned accents — do not drift

These are locked by `src/presentation/WildbloomDiscoveryController.ts` (`WILDBLOOM_SPOTS`). They are authoritative gameplay values; the Wildbloom magical-family art must match them exactly. They intentionally overlap the `forest`/`arcane` ranges.

## Family usage by target file

- `farm_village_tile_targets.json` — `forest`, `wood_leather`, plus `ruins`/`ui_neutral` on the village-shop targets (village stone/UI colors are out of this farm lock's scope and locked when village art lands).
- `farm_water_shoreline_targets.json` — `arcane`, `forest`, `wood_leather`, `metal`.
- `farm_vegetation_targets.json` — `forest`, `wood_leather`, `metal`, `arcane`.
- `farm_props_targets.json` — `wood_leather`.
- `wildbloom_magical_environment_target.json` — `arcane`, `forest` + the pinned accents above.

The machine-readable source is [`farm_palette.json`](farm_palette.json), validated by `npm run validate:visual-targets`.

# 2026-07-19 — Game feel + purposeful interactions (play-feedback milestone)

- Author/branch: Claude Code (repo agent), `feel/game-feel-purposeful-interactions`, driven by four pieces of real play feedback from the repo owner and a child playtester: the hero felt slow, the camera read as laggy, too many school questions with no direction, and repeatable purposeless interactions (the slime never disappeared).
- Scope, in four bounded parts:
  1. **Movement/camera feel** — max speed `250 → 350` world px/sec, light per-frame velocity smoothing (0.3 lerp with an exact-zero stop snap), camera follow lerp `0.12 → 0.3`, footstep interval `380 → 300` ms, Mage walk frameRate `8 → 10` (stride distance per animation frame stays ~constant, avoiding foot-slide). All values centralized in `src/movementTuning.ts` with unit-tested playtested bounds.
  2. **Permanent Practice Slime defeat** — new persisted `practiceSlimeDefeated` quest flag via the existing open `questFlags` record (no schema version bump; missing flag ⇒ `false`, so every old save loads with the slime present). On three-hit completion at any quest step: flag set and saved immediately, slime/pips hidden, interaction target removed; the first-defeat combat prompt still opens. `PracticeSlimeEncounterController.reset()` is now a documented test-only hook; `retire()` handles the permanent-removal presentation. Soft-lock guards: crop completion routes past `find-slime` when the slime is already defeated, and `fromSave` normalizes a `find-slime` save with the flag to `return-to-mira`.
  3. **Post-purpose pacing** — once an interactable's quest purpose is fulfilled, repeats show short rotating pre-reader-friendly flavor toasts (`src/data/flavor.ts`) instead of opening a prompt. The crop patch and Mira end their toasts with "ACTION again to practice!" — a second press within 2s opens the optional prompt (Mira rotates combat/farm/quest contexts, replacing the retired slime as the combat-practice tap). Post-errand sprouts are pure flavor. First-time/quest-relevant interactions are unchanged. Learning never gates adventure — nothing requires answering, before or after.
  4. **Objective direction** — `FarmQuestSystem.currentObjectiveTarget()` pure step→target mapper (unit-tested per step, `null` when all errands are complete); a bouncing gold chevron above the current target and a screen-edge arrow while it is off-camera, both plain Graphics respecting the e2e "first Container" discovery convention. The text banner and Mira pulse remain.
- Principal files: `src/movementTuning.ts` (new), `src/data/flavor.ts` (new), `src/scenes/WorldScene.ts`, `src/scenes/PolishedWorldScene.ts`, `src/presentation/PracticeSlimeEncounterController.ts`, `src/presentation/HeroPresentationController.ts`, `src/systems/FarmQuestSystem.ts`, `src/data/quests.ts` (one new progress string), `tests/unit/movementTuning.test.ts` (new), `tests/unit/FarmQuestSystem.test.ts` (new), `tests/game-feel-direction.spec.ts` (new), plus targeted updates to `tests/practice-slime-encounter.spec.ts`, `tests/vertical-slice.spec.ts`, and `tests/system-foundations.spec.ts` where they asserted the old speed/reset behavior.
- Verification: `npm run check` green; `npm run test:unit` green (108 tests, including the new tuning-bounds, flag-migration/soft-lock, purpose-predicate, and objective-target suites); full local Playwright suite green (see PR for the exact run summary); browser evidence in `docs/playtests/2026-07-19-game-feel-milestone/`.
- Compatibility: saves fully backward compatible (open `questFlags` record; no version bump; migration behavior unit-tested). No dependency, curriculum, mastery, reward, profile-ID, or map changes. Flavor-line copy is new player-facing text written to the milestone prompt's constraints — flagged for ChatGPT wording review rather than treated as approved story canon.
- Remaining risk: feel values are tuned in-browser, not physically iPad-validated; the 2s opt-in window and flavor copy may warrant kid-testing adjustment; the objective marker's world positions assume the current farm map object coordinates.

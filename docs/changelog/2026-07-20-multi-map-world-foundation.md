# 2026-07-20 — Multi-map world foundation (M2)

- Author/branch: Claude Code (repo agent), `world/multi-map-foundation`, executing Build Prompt 02 (world-building foundation) on top of the merged game-feel milestone (PR #104). This is the **M2** milestone that the approved Mossheart Ruins (Zone #4) design spec declares as its prerequisite.
- Scope, four phases (one commit each, each green):
  1. **Map registry + generalized WorldScene.** New `src/data/maps.ts` holds a `MapDefinition` registry (`tiledKey`, `jsonPath`, `displayName`, `tilesets`, `musicKey`, named world-px `spawns`, `defaultSpawn`, `collisionGids`) plus `validateMapRegistry()`, which cross-checks the registry against the **real committed map JSONs** (unique keys/paths, spawns inside world bounds, declared tilesets present, every exit's target resolving). `WorldScene` accepts `{ mapId?, spawnId? }` and now reads every map specific from the registry — the farm tilemap key, tileset wiring, collision GIDs, music key, and world-bounds derivation are all gone from the scene. `PreloadScene` loads map JSONs in a registry loop with stable cache keys. A map-name entry banner (~1.5s fade) plays on every scene start.
  2. **Exits and transitions.** Tiled convention: Objects-layer `type: "exit"` rectangles carrying `targetMap` / `targetSpawn` (plus a documented-but-unused `requiresQuestFlag`). Exits stay in the map files rather than the registry so each map remains the single source of truth for its own geometry. Trigger locks input, resets the joystick, fades out 300ms, restarts the scene at the target spawn, fades in, and banners; a `transitioning` flag guards re-entry and cross-restart state resets in `init()`.
  3. **Wildbloom Woods.** New `public/maps/wildbloom-woods.json` (20×14) built **only from already-approved/committed art** (the approved grass/dirt terrain-proof masters and the placeholder structural tileset the farm already uses): collidable tree border with a two-tile gate gap, interior clumps shaping a central clearing, a dirt gate path, walk-behind `Decor` trees and a signpost. Two quest-free interactables on the existing id/handler registry: `whispering-flower` (rotating Wildbloom-theme flavor) and `mossy-stone` (flavor plus the Prompt-01 second-ACTION practice opt-in, exploration context). The farm gained a matching east gate (road extended through the border using approved dirt masters, border collision cleared, signpost, `GateToWoods` exit); re-running `compose-terrain-proof-tileset.mjs` confirms the farm edit is idempotent under the deterministic map pipeline. Music reuses `bgm-farm`, documented as the accepted placeholder for this milestone.
  4. **Tests, docs.** New `docs/MAP_AUTHORING.md` is the authoring contract (Tiled layer/object conventions, registry fields, spawn placement priority, collision GIDs and the 72px body caveat, tileset/music wiring, validation, checklist, worked example).
